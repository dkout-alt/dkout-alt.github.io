窗口和句柄 Handle

因为窗口就是窗口啊！
窗口可分为客户区和非客户区
窗口可以有一个父窗口
在 Windows App 中，窗口是通过窗口句柄（HWND）来表示的
系统在创造资源时会分配内存，并返回标识这些资源的标识号

消息与消息队列

消息首先被操作系统捕获 操作系统封装，塞进消息队列

操作系统 => 消息队列 》 应用程序 == DispatchMessage ==> 操作系统 => 窗口过程(可以视为回调函数)

WinMain 函数

启动一个程序时， 插入到可执行文件中的启动代码 调用该程序的 WinMain 函数
Winmain 是Windows 程序的入口点函数 这个函数结束或返回时，应用程序结束

Windows 编程模型

实现一个程序的步骤：

  • 定义 WM 函数
  • 创建一个窗口
  • 进行消息循环
  • 编写窗口过程函数
